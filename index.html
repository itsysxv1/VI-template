Embed Builder

Visually create Discord embeds with live preview and code export.

const embed = new EmbedBuilder()
  .setTitle("Welcome!")
  .setDescription("Create Discord bots easily")
  .setColor("#0099ff");
channel.send({ embeds: [embed] });

Component Builder

Create interactive Discord components: buttons, menus, modals.

const row = new ActionRowBuilder()
  .addComponents(
    new ButtonBuilder()
      .setCustomId('primary')
      .setLabel('Primary')
      .setStyle(ButtonStyle.Primary)
  );
await interaction.reply({ content: 'Choose:', components: [row] });
